# CLV Prediction using Linear Regression

##  Dataset
Online Retail Dataset (UCI)

##  Objective
Predict Customer Lifetime Value (CLV) using Linear Regression.

##  Steps Performed
- Data Cleaning
- Feature Engineering
- Model Training
- Assumption Checking (Linearity, Homoscedasticity, VIF)
- Model Evaluation (RMSE, MAE, R²)

##  Results
Model shows good performance in predicting CLV.

##  Tools Used
- Python
- Pandas, NumPy
- Scikit-learn
- Statsmodels
- Matplotlib, Seaborn
